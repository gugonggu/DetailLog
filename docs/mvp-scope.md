# MVP Scope

## In Scope For This Step

- Initialize or verify Next.js App Router project structure
- Set up TypeScript and Tailwind CSS
- Prepare feature-based folders
- Create basic route groups
- Add placeholder pages for the requested routes
- Add a clean responsive landing page placeholder
- Add baseline documentation

## Out Of Scope For This Step

- Supabase connection
- OpenAI connection
- Database schema
- Auth implementation
- File upload implementation
- AI routine recommendation logic
- Community posting features
- Native mobile app work
- App Store or Play Store release work

## First Release Target

The first release target is a responsive web MVP deployed on Vercel.
