# 차량 도장면 보호 상태 선택 설계

## 목표

차량 등록 및 수정 화면의 모호한 `코팅 타입` 자유 입력을 이해하기 쉬운
`도장면 보호 상태` 선택 목록으로 변경한다.

## 동작

- 선택 항목은 `잘 모르겠음`, `코팅 없음`, `왁스`, `실런트`,
  `유리막·세라믹 코팅`, `PPF`, `기타`로 제한한다.
- 신규 차량의 기본값은 `잘 모르겠음`이다.
- 기존 차량에 선택 목록에 없는 값이 저장되어 있으면 수정 화면에서 해당 값을
  임시 선택 항목으로 표시해 기존 데이터를 잃지 않는다.
- DB 컬럼과 내부 식별자 `coating_type`, `coatingType`은 호환성을 위해 유지한다.
- 차량 상세 화면의 표시 라벨도 `도장면 보호 상태`로 변경한다.

## 검증

- Zod 스키마가 목록 내 값을 허용하고 임의의 신규 값을 거부하는지 테스트한다.
- 기존 값 호환을 위한 선택 항목 생성 함수를 테스트한다.
- 전체 테스트, 린트, 프로덕션 빌드를 실행한다.
